## 🌐 访问你的 Pages 页面

### 步骤 1：启用 GitHub Pages

1. 访问：https://github.com/BiQing-hc/Claude_Video_Ai/settings/pages
2. 在 **Source** 下选择：

   - Folder: `/ (root)`
3. 点击 **Save**

启用后，你的页面将在以下地址可用：
